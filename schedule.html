💡 How it works

UWA CAS Import

To import from UWA's Class Allocation System:

  1. Log in to cas.uwa.edu.au
  2. Go to My Timetable
  3. Export as .ics file
  4. Click Import Timetable above
Your CAS login credentials are never stored by SmartMeet.
